Welcome to WAID (What Am I Doing) the app that allows you to share moments at the time of experience.

After downloading the Android (WAID) What Am I Doing ? application Enter an email address, a password, firstname, lastname and you can start sharing moments.

If some one has already shared a stream with you then you were automatically registered and you should have received your password in an email. Click on here to Request a new password

Description of Actions:

  • Start Sharing  This is used to initiate and stop a live stream.
  • Start Camera  This is used to start the camera
  • Send Location  This is used to transmit you current location. Viewers of your stream will be able to see where you are on a map.
  • Whoes Watching  This will return information about those you have invited to view your stream

Usage:

  • Click on the "Start Sharing" and the "Start Camera"
  • On the status bar you should see
  • When all the buttons are enabled you are ready to invite people to view your stream.
  • You can send an email, post on facebook or tweet on twitter. Using the icons on the menu bar
  • Once you stop sharing then the stream is no longer active.
  • If you start sharing again you will need to re-invite so that they can view the new live strem
